So, what the hell is this?

Whenever you do any kind of web project, you'll have to have some kind of "backend" and a "frontend". The backend is how you get the stuff you want your site to see, and the front end is how you see it (In a super broad sense)

So, what are Flask and Python?

Python is a programming language that's great for everything from small scale projects to large deployments. It's one of the most popular programming languages, and is very easy to get started with.

Flask is a powerful web microframework that allows you to create a simple web applicaiton very easily. It does this by creating "routes" or "endpoints" you can use to easily organize and distribute data.

What do I need to get started?
